
# SmartVote AI Global 🌍

Това е универсално приложение за гласуване, създадено да обедини света и да даде равен шанс на всеки, включително хора с тежки увреждания.

### 🌟 Основни характеристики:
- **Всичко в едно:** Няма отделни менюта. Един екран за всички.
- **Сляпо-глухи достъпност:** Използва тактилни импулси (вибрация), за да потвърди вота чрез допир.
- **ИИ Гласов анализ:** Разпознава местоимения на различни езици (Аз, I, Yo, Ich).
- **Глобален обхват:** Поддържа български, английски, испански и немски в един файл.

### 🛠 Инсталация:
1. `pip install flet SpeechRecognition PyAudio`
2. `python main_global.py`
